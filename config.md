<summary><b> =EN= Notes. Click to expand</b></summary>
The configuration options are:<br>
<br>
- <b>"Column alignment"</b>: Either:<br>
&nbsp&nbsp&nbsp- <i><b>"Start"</b></i>: text in column is left aligned (or right aligned for right-to-left scripts)<br>
&nbsp&nbsp&nbsp- <i><b>"Center"</b></i>: text in column is center aligned<br>
<br>
- <b>"Language</b>: String. If an empty string is specified, the program will attempt to find a translation file for the language specified by Anki. Otherwise, specify the file name without the extension (for example, "ru" for Russian) located in the add-on's folder (Example location of this file: "...Anki2\addons21\874215009\advancedbrowser\localization\ru.lng" You can open the location of this add-on from the Anki program: Tools -> Add-ons -> Select the name of this add-on -> Click the "View files" button -> Go to the "advancedbrowser\localization" directory). If your language is unavailable, create a copy of the "en.lng" file with a new name (for your language), open the file in a text editor, and translate only the text between the === blocks.<br>
<br>
- <b>"Show internal fields"</b>: Boolean (i.e. "true" or "false"). Enable even more columns that map to database fields related to the cards and notes (dev option)<br>
<br>
- <b>"Table content"</b>: Either:<br>
&nbsp&nbsp&nbsp- <i><b>"No interaction"</b></i>: the table content can't be interacted with. This makes the Advanced Browser behave like the regular Anki browser.<br>
&nbsp&nbsp&nbsp- <i><b>"Selectable"</b></i>: the table content can be selected. All changes will be ignored. This is useful because this allows you to quickly copy field contents (instead of the regular procedure where you have to do all these steps: select the note, move the mouse to the field, select its content, then copy it).<br>
&nbsp&nbsp&nbsp- <i><b>"Editable"</b></i>: Allow to edit some values directly in the table. Not all.<br>

<summary><b>=PT-BR= Notas.</b></summary>
As opções de configuração são:<br>
<br>
- <b>"Column alignment"</b>: Uma das opções:<br>
&nbsp&nbsp&nbsp- <i><b>"Start"</b></i>: o texto na coluna é alinhado à esquerda (ou à direita para sistemas de escrita da direita para a esquerda)<br>
&nbsp&nbsp&nbsp- <i><b>"Center"</b></i>: o texto na coluna é alinhado ao centro<br>
<br>
- <b>"Language"</b>: String. Se uma string vazia for especificada, o programa tentará encontrar um arquivo de tradução para o idioma definido no Anki. Caso contrário, especifique o nome do arquivo sem a extensão (por exemplo, "pt" para português), localizado na pasta do add-on (Exemplo de localização deste arquivo: "...Anki2\addons21\874215009\advancedbrowser\localization\ru.lng" Você pode abrir a localização deste complemento no programa Anki: Ferramentas -> Complementos -> Selecione o nome deste complemento -> Clique no botão "Procurar arquivos" -> Acesse o diretório "advancedbrowser\localization"). Se o seu idioma não estiver disponível, crie uma cópia do arquivo <i><b>"en.lng"</b></i> com um novo nome (para o seu idioma), abra-o em um editor de texto e traduza apenas o texto entre os blocos "===".<br>
<br>
- <b>"Show internal fields"</b>: Booleano ("true" ou "false"). Ativa colunas adicionais que correspondem a campos internos do banco de dados relacionados às cartas e notas (opção para desenvolvedores).<br>

- <b>"Table content"</b>: Uma das opções:<br>
&nbsp&nbsp&nbsp- <i><b>"No interaction"</b></i>: o conteúdo da tabela não pode ser interagido. Isso faz com que o Advanced Browser se comporte como o navegador padrão do Anki.<br>
&nbsp&nbsp&nbsp- <i><b>"Selectable"</b></i>: o conteúdo da tabela pode ser selecionado. Todas as alterações serão ignoradas. Isso é útil para copiar rapidamente o conteúdo dos campos (em vez do procedimento comum: selecionar a nota, mover o cursor até o campo, selecionar o conteúdo e copiá-lo).<br>
&nbsp&nbsp&nbsp- <i><b>"Editable"</b></i>: permite editar alguns valores diretamente na tabela. Não todos.<br>
